# Test cases

## FIFO base

1. Покупка 100 шт. по 10
2. Покупка 50 шт. по 12
3. Продажа 120 шт. по 15

Ожидаемый результат:
- списание 100 шт. по 10;
- списание 20 шт. по 12;
- остаток 30 шт. по 12.

## Проверки

- Продажа больше остатка должна показывать ошибку.
- Списание ЦБ должно списывать себестоимость по FIFO.
- Комиссия должна отдельно влиять на расчет с комиссией.
- Расчет без комиссии не должен изменяться от комиссии.
- При фильтре по периоду должны корректно учитываться начальные остатки.
