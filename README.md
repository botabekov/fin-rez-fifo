# Fin Rez FIFO

Одностраничный браузерный MVP для анализа финансового результата по операциям с ценными бумагами с применением FIFO и расчетом KZT equivalent.

## Что делает

- Загружает Excel/CSV с движением инструментов.
- Группирует операции по инструментам.
- Считает себестоимость, сумму продажи и прибыль/убыток по FIFO.
- Показывает расчет без комиссии и с учетом комиссии.
- Отдельно отображает:
  - движение инструментов;
  - дивиденды;
  - РЕПО;
  - остатки ЦБ.
- Позволяет сформировать отдельные формы финансового результата.

## Как запустить

Откройте `index.html` в браузере.

Если браузер блокирует локальные файлы, запустите простой локальный сервер:

```bash
python -m http.server 8000
```

После запуска откройте:

```text
http://localhost:8000
```

## Важно

Это MVP/прототип для проверки логики и пользовательского сценария.  
Не используйте результат как официальный налоговый расчет без дополнительной проверки.

## Публикация через GitHub Pages

1. Создайте репозиторий `fin-rez-fifo`.
2. Загрузите файлы из этого архива.
3. В GitHub откройте **Settings → Pages**.
4. Source: `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Сохраните настройки.
